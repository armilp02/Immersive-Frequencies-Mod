### What's it about?

This mod adds walkie talkies that let you talk to other players across the map. Perfect for survival servers, roleplay, or just staying in touch while exploring.

### Requirements

*   [**Simple Voice Chat**](https://www.curseforge.com/minecraft/mc-mods/simple-voice-chat) - Required
*   [**Fabric API**](https://www.curseforge.com/minecraft/mc-mods/fabric-api) - Required on Fabric version

### How to use

1.  Craft a walkie talkie
2.  Set a frequency (In the GUI, Default key is `R`)
3.  Hold the talk key to broadcast
4.  Others on the same frequency will hear you

For now the range of the walkie is not configurable (1000 blocks)

That's it. Have fun!
